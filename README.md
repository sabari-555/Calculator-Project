🧮 Responsive Calculator Web Application

A modern calculator web application built with HTML, CSS, and JavaScript. The application features a responsive user interface, Dark/Light mode support, and comprehensive unit testing using Jest.

Key Features:
• Basic arithmetic operations
• Responsive design for mobile, tablet, and desktop devices
• Dark and Light theme toggle
• Clean and intuitive user interface
• Error handling and input validation
• Jest unit testing with 100% code coverage

Tech Stack:
• HTML5
• CSS3
• JavaScript (ES6)
• Jest
